.NET ------------------------------------------------------------

Criar projeto .NET:
    dotnet new sln
    dotnet new webapi -o api
    dotnet sln add api

Run:
    dotnet watch run
    npm start

Migrations ------------------------------------------------------------

Criar migration:
    dotnet ef migrations add ExtendedUserEntity
    dotnet ef migrations add LikeEntityAdded
    dotnet ef database update (implementa até última migration, mesmo efeito de rodar a api)

Desfazer migration m:
    dotnet ef database update 'nome m-1'

Remove último migration:
    dotnet ef migrations remove

Recriar as tabelas:
    dotnet ef database drop
    dotnet ef database update (implementa até última migration, mesmo efeito de rodar a api)
    
Angular ------------------------------------------------------------

Criar projeto Angular:
    npm install -g @angular/cli
    ng new client


Angular Component:
    ng g c nav --skip-tests

Angular Service:
    ng g s account --skip-tests

Angular Guard:
    ng g guard auth --skip-tests

Angular Module:
    ng g m shared --flat

Angular Interceptor:
    ng g interceptor error --skip-tests
