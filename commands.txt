Run:
    dotnet watch run
    npm start

Refazer tabelas:
    dotnet ef database drop
    dotnet ef database update

Gerar migration:
    dotnet ef migrations add ExtendedUserEntity

Desfaz migration m:
    dotnet ef migrations revert 'nome m-1'

Remove último migration:
    dotnet ef migrations remove

Atualiza tabelas da database conforme migrations:
    dotnet ef migrations remove

Angular Component:
    ng g c nav --skip-tests

Angular Service:
    ng g s account --skip-tests

Angular Guard:
    ng g guard auth --skip-tests

Angular Module:
    ng g m shared --flat

Angular Interceptor:
    ng g interceptor error --skip-tests
